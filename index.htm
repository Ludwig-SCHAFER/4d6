🎲 🎲 🎲 🎲 🎲 4d6 drop lowest and retry until you roll only 6s

Rules

  1. ⚀ Roll 4 6-sided dices.
  2. ⚁ Reroll the lowest dice.
  3. ⚂ Retry until there are only 6s.

Comments

  • ⚃ Dices icons used on this webpage are unicode characters.
  • ⚄ Edit the SIZE variable in the script to roll more or less dices.
  • ⚅ Good luck.

Legend

Minimal value is displayed in red.
The rerolled value is displayed in gold.
